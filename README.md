# Ashtar

ChickenLive
-------------
>資展國際全端工程師養成班-結訓專案。  
>是由我擔任組長並與團隊一同完成的專案，主題為即時聊天互動平台，支援1對1配對與視訊、語音交友。  
>分為前台專案(Vue.js+web api)與後台專案(ASP.NET CORE MVC)。

前台(Vue.js+web api)
---
>我負責的工作項目:
>>使用Phaser3撰寫整個遊戲架構
>>前端切板與主視覺設計
>>合作廠商廣告的web api接口
>>與負責websocket、webRTC部分的團隊成員對接並完成專案整合，以實現多人遊戲。
>>與負責vue.js框架的團隊成員合作將專案移植為vue3的專案。
